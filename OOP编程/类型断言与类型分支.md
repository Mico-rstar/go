### 类型断言
语法
```go
v, ok := x.(T)
```

```go
var w io.Writer
w = os.Stdout
f := w.(*os.File)   // success
c := w.(*bytes.Buffer)  // panic
```
直接panic并不是推荐做法，更好的方式如下
```go
var w = io.Writer
w = os.Stdout
if f, ok := w.(*os.File); ok {
    ...
}
```

### 类型分支
```go
func sqlQuote(x interface{}) string {
    switch x := x.(type) {
    case nil:
        return "NULL"
    case int, uint:
        return fmt.Sprintf("%d", x) // x has type interface{} here.
    case bool:
        if x {
            return "TRUE"
        }
        return "FALSE"
    case string:
        return sqlQuoteString(x) // (not shown)
    default:
        panic(fmt.Sprintf("unexpected type %T: %v", x, x))
    }
}

```
