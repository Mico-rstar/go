在 Go 语言中，占位符主要用于格式化输出，常见于 `fmt` 包中的函数（如 `Printf`、`Sprintf`、`Fprintf` 等）。以下是常用的占位符分类和示例：

---

## 1. 通用占位符
| 占位符 | 说明 |
|--------|------|
| `%v` | 按值的默认格式输出（适合任何类型） |
| `%+v` | 类似 `%v`，但输出结构体时会包含字段名 |
| `%#v` | 值的 Go 语法表示（可读性更强的完整结构） |
| `%T` | 输出值的类型 |
| `%%` | 字面百分号（非占位符） |

**示例：**
```go
type User struct {
    Name string
    Age  int
}
u := User{"Alice", 25}
fmt.Printf("%v\n", u)   // {Alice 25}
fmt.Printf("%+v\n", u)  // {Name:Alice Age:25}
fmt.Printf("%#v\n", u)  // main.User{Name:"Alice", Age:25}
fmt.Printf("%T\n", u)   // main.User
```

---

## 2. 布尔值占位符
| 占位符 | 说明 |
|--------|------|
| `%t` | 输出 `true` 或 `false` |

**示例：**
```go
fmt.Printf("%t", true) // true
```

---

## 3. 整数占位符
| 占位符 | 说明 |
|--------|------|
| `%b` | 二进制 |
| `%d` | 十进制 |
| `%o` | 八进制 |
| `%x` | 十六进制（小写字母） |
| `%X` | 十六进制（大写字母） |
| `%c` | 对应的 Unicode 字符 |
| `%q` | 带单引号的字符（如 `'a'`） |

**示例：**
```go
n := 65
fmt.Printf("%b\n", n) // 1000001
fmt.Printf("%d\n", n) // 65
fmt.Printf("%o\n", n) // 101
fmt.Printf("%x\n", n) // 41
fmt.Printf("%c\n", n) // A
fmt.Printf("%q\n", n) // 'A'
```

---

## 4. 浮点数与复数
| 占位符 | 说明 |
|--------|------|
| `%f` | 浮点数（默认精度 6） |
| `%e` | 科学计数法（如 `1.23e+01`） |
| `%E` | 科学计数法（大写 E） |
| `%g` | 根据情况选择 `%f` 或 `%e`（更紧凑） |
| `%G` | 根据情况选择 `%f` 或 `%E` |

**示例：**
```go
f := 123.456
fmt.Printf("%f\n", f)  // 123.456000
fmt.Printf("%.2f\n", f) // 123.46（保留2位小数）
fmt.Printf("%e\n", f)  // 1.234560e+02
```

---

## 5. 字符串与字节切片
| 占位符 | 说明 |
|--------|------|
| `%s` | 直接输出字符串/字节切片 |
| `%q` | 带双引号的字符串（转义特殊字符） |
| `%x` | 十六进制（小写，每字节两个字符） |
| `%X` | 十六进制（大写） |

**示例：**
```go
s := "Hello"
fmt.Printf("%s\n", s)  // Hello
fmt.Printf("%q\n", s)  // "Hello"
fmt.Printf("%x\n", s)  // 48656c6c6f（十六进制）
```

---

## 6. 指针
| 占位符 | 说明 |
|--------|------|
| `%p` | 指针的十六进制表示（带 `0x` 前缀） |

**示例：**
```go
x := 42
fmt.Printf("%p\n", &x) // 0xc0000120a0（地址随运行变化）
```

---

## 7. 宽度与精度控制
- `%5d`：整数最小宽度为5（右对齐）
- `%-5d`：左对齐
- `%05d`：补零至5位
- `%8.2f`：总宽度8，保留2位小数

**示例：**
```go
fmt.Printf("|%5d|\n", 123)    // |  123|
fmt.Printf("|%-5d|\n", 123)   // |123  |
fmt.Printf("|%05d|\n", 123)   // |00123|
fmt.Printf("|%8.2f|\n", 123.4) // |  123.40|
```

---

## 8. 切片与数组
切片和数组默认可用 `%v` 输出，但也可用 `%#v` 显示详细类型信息：
```go
arr := [3]int{1, 2, 3}
fmt.Printf("%v\n", arr)  // [1 2 3]
```

---

## 简单总结表
| 类型 | 常用占位符 |
|------|------------|
| 通用 | `%v`, `%+v`, `%#v`, `%T` |
| 布尔 | `%t` |
| 整数 | `%d`, `%b`, `%x`, `%o`, `%c` |
| 浮点数 | `%f`, `%e`, `%g` |
| 字符串 | `%s`, `%q`, `%x` |
| 指针 | `%p` |

需要更多细节可查阅 Go 官方文档：https://pkg.go.dev/fmt。
