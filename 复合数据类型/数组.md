### 数组
数组是固定长度相同类型元素集合

### 数组基本用法
```Go
// 声明时初始化
var arr1 [3]int = [3]int{1, 2, 3}

// 简短声明
arr2 := [3]int{1, 2, 3}

// 指定索引初始化
arr := [5]int{a: 10, 3:10} // [0, 10, 0, 10, 0]

const (
    USD Currency = iota // 美元
    EUR                 // 欧元
    GBP                 // 英镑
    RMB                 // 人民币
)

// 使用[...]由初始化元素推断数组长度
symbol := [...]string{USD: "$", EUR: "€", GBP: "￡", RMB: "￥"}
```

### 遍历数组
```Go
// c风格
for i := 0; i < len(arr); i++ {
    fmt.Printf("索引%d: %s\n", i, arr[i])
}

// 使用range
for index, value := range arr {
    fmt.Printf("索引%d: %s\n", index, value)
}
```


### 特性 
1. 默认值拷贝
    ```Go
    // 函数参数也是值拷贝
    func modifyArray(arr [3]int) {
        arr[0] = 999  // 不影响原数组
    }
    ```

2. 长度是类型一部分
   ```Go
    var a [3]int
    var b [5]int

    // a = b  // 编译错误：类型不匹配 [3]int ≠ [5]int
   ```

3. 多维数组
    ```Go
    // 二维数组声明和初始化
    var matrix1 [2][3]int
    matrix1[0] = [3]int{1, 2, 3}
    matrix1[1] = [3]int{4, 5, 6}

    // 简短声明
    matrix2 := [2][3]int{
        {1, 2, 3},
        {4, 5, 6},
    }

    // 遍历二维数组
    for i := 0; i < len(matrix2); i++ {
        for j := 0; j < len(matrix2[i]); j++ {
            fmt.Printf("matrix[%d][%d] = %d\n", i, j, matrix2[i][j])
        }
    }
    ```
4. 可比较
    如果数组中的元素是可比较的，那么数组是可比较的
    ```Go
    a := [2]int{1, 2}
    b := [...]int{1, 2}
    c := [2]int{1, 3}
    fmt.Println(a == b, a == c, b == c) // "true false false"
    d := [3]int{1, 2}
    fmt.Println(a == d) // compile error: cannot compare [2]int == [3]int
    ```
